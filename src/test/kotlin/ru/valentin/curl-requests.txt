Загрузка файла-вложения для задачи
POST /api/tasks/{taskId}/attachment
curl -s -v -X POST \
  http://localhost:8080/api/tasks/1/attachment \
  -H "Content-Type: multipart/form-data" \
  -F "file=@./att"

Получение вложения по идентификатору задачи
GET /api/tasks/{taskId}/attachment/download
curl -s -v -X GET \
  "http://localhost:8080/api/tasks/1/attachment/download" \
  -H "Accept: application/octet-stream, application/json" \
    --output "output_file" \

Изменение существующей задачи
PUT /api/tasks/{taskId}

Удаление задачи по идентификатору
DELETE /api/tasks/{taskId}

Создание нового тега
POST /api/tags

Изменение существующего тега
PUT /api/tags/{tagId}

Удаление тега по идентификатору (вместе с его задачами)
DELETE /api/tags/{tagId}

Получение списка задач за заданную дату с сортировкой по уровню приоритета
GET /api/tasks/by-date
curl -s -v -X GET \
  "http://localhost:8080/api/tasks/by-date?date=2025-08-21&page=0&size=5&sort=asc" \ | jq

Получение тега по идентификатору с задачами, сортированными по приоритету
GET /api/tags/{id}/tasks
curl -s -v -X GET \
  "http://localhost:8080/api/tags/1/tasks" \ | jq

Получение всех тегов, у которых есть задачи
GET /api/tags/with-tasks
curl -s -v -X GET \
  "http://localhost:8080/api/tags/with-tasks" \ | jq

Получение списка типов задач с уровнем приоритета
GET /api/task-types
curl -s -v -X GET \
  "http://localhost:8080/api/task-types" \ | jq
